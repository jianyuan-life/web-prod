## TL;DR

**L2 報的 27 處屬 grep `15` 純數字粗計、虛報** — 含 15 分鐘 / 15:00 / 12-15 歲 / 5/15 / 2015 等無關時間數字。

**真實違反命中**(語意 = 對外宣稱 15 系統 / 十五系統 / 十五套):
- C 何紀萳 64b15504 = **7 處**
- C 何宥諄 d143f949 = **7 處**

但 **7 處 / 報告皆為對外文案、且 prompt 內部仍命「15 系統」、frontend 對外已寫「14 套」、Production code 內外不一致 = L2 P0 認定成立**。

**根因**:`prompts/c_plan_v2.ts` 兩條鐵律相互矛盾:
- L332-343「14 系統最少引用次數鐵律」(v5.10.64 加、對外清零)
- L1238/1240/1242/1283/1299/1411/1818/1832 仍寫「15 系統 / 十五系統 / 十五套 / 15系統排盤速覽」

AI 自然取「指令數量更多 / 範例更具體」那邊 = **14 鐵律一句話 vs 15 系統 8+ 處範例 + 矩陣骨架 + 附錄 hardcode**、必輸給 15。

### 1.3 L2 「27 處」的真相

L2 用 `grep -c '15'` 粗計、含以下無關命中(非違反):
- 「**15** 分鐘」(運動 / 散步 / 親子陪伴 / 靜坐)× 12+ 次
- 「**15**:00-17:00」(時段)× 4 次
- 「12-**15** 歲」(青春期段落)× 1 次
- 「5/**15** 前(國曆)」(日期)× 2 次
- 「**2015**-2033」(吠陀大限年份)× 2 次
- 「±**15**% 累積誤差」(派別差異註)× 1 次
- 「+**15** 分」(生肖月運)× 1 次
- 「第 **15** 號」(塔羅惡魔牌編號、塔羅 22 大牌之一)× 1 次

**L2 報虛、但 P0 結論成立**(根因確診同一條:prompt 內外不一致)。

### 修法 B:Frontend 後處理 sanitize(推薦、雙保險、立即生效舊報告)

**LOC**:`app/report/[token]/page.tsx` 加 ~15 行
**風險**:極低、已有 cleanFinalReport / parseStructuredContent 框架可加

#### B-1 sanitize 函式

```typescript
// 加在 cleanFinalReport 前後
function sanitizeFifteenToFourteen(content: string): string {
  return content
    // 對外文案 hardcode replace
    .replace(/十五系統交叉驗證/g, '十四系統交叉驗證')
    .replace(/十五個系統交叉/g, '十四個系統交叉')
    .replace(/十五個系統都/g, '十四個系統都')
    .replace(/十五張底片/g, '十四張底片')
    .replace(/十五個系統看/g, '十四個系統看')
    .replace(/十五套命理系統/g, '十四套命理系統')
    .replace(/十五系統交叉驗證的全貌/g, '十四系統交叉驗證的全貌')
    .replace(/15系統排盤速覽/g, '14系統排盤速覽')
    .replace(/(\b)15 套系統/g, '$114 套系統')
    .replace(/(\b)15 個系統/g, '$114 個系統')
    .replace(/(\b)15 系統(?!排盤速覽)/g, '$114 系統')
    // 矩陣表格欄位刪「南洋」(對外 14)
    // 註:此處留 prompt 控制、frontend 不動表格結構避破壞 markdown
}

// 在 page.tsx render content 前 wrap:
const safeContent = sanitizeFifteenToFourteen(rawAiContent)
```

**好處**:
- 修補無需重 generate(舊報告立即合規)
- 跟 prompt 修補 A 互補(prompt 漏寫時兜底)

**風險**:
- 章節編號「## 十五、給你的一句話」會被誤改 → 用 lookahead 排除「十五、」
- 「±5%~15% 累積誤差」中的 15 不能改 → 已用 \b 邊界 + 系統字眼 anchor 排除

#### B-2 安全 regex(避免誤傷時間 / 日期 / 派別誤差)

```typescript
.replace(/十五(?=[個系套][系統套個])/g, '十四')   // 只改後接「系/套/個系統」
.replace(/15(?=\s*[套個]?\s*系統)/g, '14')        // 只改後接「系統」
.replace(/15系統排盤速覽/g, '14系統排盤速覽')      // hardcode 詞精準改
// 排除:十五、(編號) / 15:00 / 15 分鐘 / ±15% / 第 15 號 / 5/15 / 2015 / 12-15 歲
```
